<h1>Landing Page - 🧴 Champú Cabello de Ángel</h1>
<h2>🚀Descripción del proyecto</h2>
Este proyecto consiste en una landing page orientada a la venta de un producto cosmético (champú), diseñada para destacar sus beneficios, generar interés en el usuario y fomentar la conversión mediante un llamado a la acción claro.
La página presenta el producto, sus beneficios, ingredientes naturales y una propuesta de valor enfocada en el cuidado capilar saludable.

---

<h2>🧩Problema que resuelve</h2>
En el mercado de productos capilares existe:
<br><br>
<ul>
  <li>Saturación de opciones sin diferenciación clara.</li>
  <li>Falta de comunicación efectiva de beneficios reales.</li>
  <li>Páginas poco optimizadas para conversión.</li>
</ul>
Este proyecto resuelve estos problemas mediante:
<br><br>
<ul>
  <li>💡 <b>Comunicación clara de beneficios del producto (brillo, suavidad, salud capilar).</b></li>
  <li>🌿 <b>Enfoque en ingredientes naturales como propuesta de valor.</b></li>
  <li>🎯 <b>Estructura optimizada para conversión (CTA visible y directo).</b></li>
  <li>📱 <b>Diseño responsivo para mejorar la experiencia en cualquier dispositivo.</b></li>
</ul>

---

## 🛠️ Stack tecnológico
El desarrollo se realizó utilizando:
<ul>
  <li><strong>HTML5</strong> – Estructura semántica de la landing.</li>
  <li><strong>CSS3</strong> – Estilización, layout y diseño responsivo.</li>
  <li><strong>JavaScript (Vanilla)</strong> – Interacciones básicas (botones, comportamiento UI).</li>
  <li><strong>Netlify</strong> – Deploy y hosting.</li>
</ul>

---

## ⚙️ Decisiones técnicas
<ul>
  <li>Uso de <b>estructura de landing page (hero + CTA + beneficios)</b> para maximizar conversión.</li>
  <li>Implementación de un <b>CTA principal destacado</b> (“Quiero un cabello natural y brillante”).</li>
  <li>Diseño <b>centrado en el producto</b> con imágenes prominentes.</li>
  <li>Aplicación de <b>paleta de colores emocional (rosas y verdes)</b> para transmitir naturalidad y cuidado.</li>
  <li>Enfoque <b>mobile-first</b> para asegurar accesibilidad.</li>
  <li>Uso de <b>JavaScript sin frameworks</b> para mantener bajo peso y simplicidad.</li>
</ul>

---

## 🧗 Retos encontrados
<ul>
  <li>🎨 <b>Equilibrio visual</b> entre estética y claridad del mensaje.</li>
  <li>📱 <b>Diseño responsivo</b> en secciones con imágenes grandes.</li>
  <li>🎯 <b>Jerarquía de contenido</b> para guiar al usuario hacia el CTA.</li>
  <li>⚡ <b>Optimización de carga</b> de imágenes del producto.</li>
  <li>🧩 <b>Construcción de una narrativa de venta efectiva</b> en una sola página.</li>
</ul>
